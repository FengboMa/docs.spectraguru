## Introduction

The Analytics Page is for extracting useful information from processed spectra using general analytics and machine learning methods. It includes average and interval plots, derivative analysis, FFT, correlation heatmaps, peak tools, clustering, dimensionality reduction, and supervised classification.

## General Operation

To analyze a particular set of spectra, first upload and proccess the data. Then, in the processing page, ensure that the spectra of interest are listed under the section labeled "Select Spectra you want to visualize." These are the spectra that will be used for analysis. Click the "x" next to unwanted spectra to remove them from the list.

Once the data of interest have been selected, go to the Analytics page and select the desired type of analysis using the drop-down menu. For supervised classification, assign labels on the Data Upload page before running the model. For further details, see the in-depth documentation for each analysis function.
