# Peak Assignment Table
{: .no_toc }

## Introduction

Peak Assignment Table provides a curated peak assignment table collection for research reference use. Open **Toolbox Page**, choose **Peak Assignment Table** from **Select tool**, and move the cursor over the displayed table to access search, full-screen view, column filtering, and other table tools.
